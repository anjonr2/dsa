Given an array nums and an integer k. An Array is called nice if and only if it contains k odd numbers.

Example
Input: nums = [1,1,2,1,1] k = 3
Output: 2
Explanation: The subarrays with 3 three odd numbers are 
[1,1,2,1]
[1,2,1,1]

Input : nums = [4,8,2] k = 1
Output : 0
Explanation : The array does not contain any odd numbers
